### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/wille430/ts-new.git
   ```
2. Cd into the directory and install dependencies with your preferred package manager:
    ```sh
    cd ./ts-new
    npm install
    ```
3. Create and install tarball:
   ```sh
   npm pack
   npm i -g ts-new-1.0.0.tgz
   ```
